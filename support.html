English

Support

ForgePlay helps Mac users prepare a local Windows game environment using user-provided GPTK/Wine runners, Steam, game libraries, and runtime installers from official sources.

Common setup steps

  • Choose a ForgePlay storage location.
  • Select a Game Porting Toolkit or Wine runner that you obtained from an official source.
  • Select the Windows Steam installer if you want ForgePlay to prepare a shared Steam prefix.
  • Scan or import Steam game libraries, then choose a game to inspect launch readiness.

Troubleshooting

  • Use the Diagnostics screen after a failed launch. ForgePlay runs local rule-based analysis first.
  • If optional AI diagnostics are enabled, review the redacted preview before running the local Apple Foundation Models diagnostic.
  • If a runtime such as Visual C++, DirectX, .NET, OpenAL, XNA, or PhysX is required, download the installer from the official vendor page and select it in ForgePlay.
  • For game-specific issues, include the game name, Steam App ID if available, macOS version, runner type/version, and the redacted support bundle.

Reporting issues

Use the GitHub repository issue tracker when available: github.com/Facta-Leopard/ForgePlay/issues. Review any logs or support bundle contents before attaching them.

지원 안내

기본 설정 순서

  • ForgePlay 저장 위치를 선택합니다.
  • 공식 출처에서 받은 Game Porting Toolkit 또는 Wine runner를 선택합니다.

문제 해결

  • 게임 실행이 실패하면 진단 화면을 사용하세요. ForgePlay는 먼저 로컬 Rule Engine으로 분석합니다.
  • 선택적 AI 진단을 켠 경우, 로컬 Apple Foundation Models 진단을 실행하기 전에 가려진 미리보기를 확인하세요.
  • Visual C++, DirectX, .NET, OpenAL, XNA, PhysX 같은 runtime이 필요하면 공식 제공자 페이지에서 설치 파일을 받은 뒤 ForgePlay에서 선택하세요.

Español

Soporte

ForgePlay ayuda a preparar un entorno local para juegos de Windows en Mac usando runners GPTK/Wine, Steam, bibliotecas de juegos e instaladores oficiales proporcionados por el usuario.

Pasos comunes de configuración

  • Elige una ubicación de almacenamiento para ForgePlay.
  • Selecciona un runner Game Porting Toolkit o Wine obtenido desde una fuente oficial.
  • Selecciona el instalador de Steam para Windows si quieres preparar un prefix compartido de Steam.
  • Escanea o importa bibliotecas de Steam y revisa la preparación de cada juego.

Solución de problemas

  • Usa la pantalla Diagnóstico después de un fallo de inicio. ForgePlay analiza primero con reglas locales.
  • Si activas el diagnóstico con IA, revisa la vista previa redactada antes de usar Apple Foundation Models localmente.
  • Si falta Visual C++, DirectX, .NET, OpenAL, XNA o PhysX, descarga el instalador desde la página oficial y selecciónalo en ForgePlay.
  • Para informar un problema, incluye el nombre del juego, Steam App ID si está disponible, versión de macOS, tipo/versión del runner y un paquete de soporte redactado.

Reportar problemas

Usa el gestor de issues de GitHub cuando esté disponible: github.com/Facta-Leopard/ForgePlay/issues. Revisa los archivos antes de adjuntarlos.

日本語

サポート

ForgePlay は、ユーザーが用意した GPTK/Wine runner、Steam、ゲームライブラリ、公式ソースの runtime インストーラを使って、Mac 上のローカル Windows ゲーム環境を準備します。

一般的な設定手順

  • ForgePlay の保存場所を選択します。
  • 公式ソースから入手した Game Porting Toolkit または Wine runner を選択します。
  • 共有 Steam Prefix を作成する場合は Windows 版 Steam インストーラを選択します。

トラブルシューティング

  • 起動に失敗した場合は診断画面を使用してください。ForgePlay はまずローカルのルール分析を実行します。

  • Visual C++、DirectX、.NET、OpenAL、XNA、PhysX が必要な場合は、公式提供元からインストーラを入手し ForgePlay で選択してください。
  • 問題を報告する場合は、ゲーム名、Steam App ID、macOS バージョン、runner の種類/バージョン、伏せ字化済みサポートバンドルを含めてください。
