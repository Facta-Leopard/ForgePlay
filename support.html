지원 안내 한국어

지원 요청에는 게임 이름, Steam App ID가 있으면 해당 ID, macOS 버전, GPTK/Wine 실행 엔진 종류, 선택한 Prefix, 최근 실행 결과, 화면에 보이는 오류 메시지를 포함하세요.

  • 문제 진단 화면에서 지원 번들을 만들 수 있으면 먼저 생성하고, 공유하기 전에 파일 내용을 직접 검토하세요.
  • 지원 번들에는 최근 실행 로그, 선택한 진단 결과, Prefix 메타데이터, 호환성 정보, 건너뛴 파일 목록, 앱 상태 요약이 포함될 수 있습니다.
  • Steam 비밀번호, Steam Guard 코드, 결제 정보, Apple ID 자격 증명, 개인 키, 관계없는 개인 파일은 보내지 마세요.
  • ForgePlay는 Apple GPTK, Steam, Microsoft Runtime, DirectX, NVIDIA PhysX, OpenAL, XNA 설치 파일을 앱 서버에서 호스팅하지 않습니다. 공식 페이지 또는 사용자가 가진 설치 파일을 사용합니다.
  • 버그 신고와 지원 요청은 GitHub Issues를 사용할 수 있습니다.

Support Guide English

When asking for support, include the game name, Steam App ID if available, macOS version, GPTK/Wine runner type, selected prefix, recent launch result, and any visible error message.

  • If ForgePlay offers a support bundle in Diagnostics, create it first and review the files before sharing.
  • A support bundle may include recent launch logs, selected diagnostics, prefix metadata, compatibility information, skipped file manifests, and app status summaries.
  • Do not send Steam passwords, Steam Guard codes, payment details, Apple ID credentials, private keys, or unrelated personal files.
  • ForgePlay does not host Apple GPTK, Steam, Microsoft Runtime, DirectX, NVIDIA PhysX, OpenAL, or XNA installers on an app server. Use official vendor pages or installers you already have.
  • Bug reports and support requests can be opened through GitHub Issues.

Guía de soporte Español

Al pedir soporte, incluye el nombre del juego, Steam App ID si existe, versión de macOS, tipo de ejecutor GPTK/Wine, Prefix seleccionado, resultado reciente de inicio y cualquier error visible.

  • Si ForgePlay ofrece un paquete de soporte en Diagnósticos, créalo primero y revisa los archivos antes de compartirlos.
  • El paquete puede incluir registros recientes, diagnósticos seleccionados, metadatos de Prefix, información de compatibilidad, listas de archivos omitidos y resúmenes de estado de la app.
  • No envíes contraseñas de Steam, códigos de Steam Guard, datos de pago, credenciales de Apple ID, claves privadas ni archivos personales no relacionados.
  • ForgePlay no aloja instaladores de Apple GPTK, Steam, Microsoft Runtime, DirectX, NVIDIA PhysX, OpenAL o XNA en un servidor de la app. Usa páginas oficiales o instaladores que ya tengas.
  • Los informes de errores y solicitudes de soporte pueden abrirse en GitHub Issues.

Support-Leitfaden Deutsch

Bitte nenne bei Support-Anfragen Spielname, Steam App ID falls vorhanden, macOS-Version, GPTK/Wine-Runner-Typ, ausgewählten Prefix, letztes Startergebnis und sichtbare Fehlermeldungen.

  • Wenn ForgePlay unter Diagnose ein Support-Paket anbietet, erstelle es zuerst und prüfe die Dateien vor dem Teilen.
  • Ein Support-Paket kann aktuelle Startlogs, ausgewählte Diagnosen, Prefix-Metadaten, Kompatibilitätsinformationen, Listen übersprungener Dateien und App-Status enthalten.
  • Sende keine Steam-Passwörter, Steam-Guard-Codes, Zahlungsdaten, Apple-ID-Zugangsdaten, privaten Schlüssel oder unbeteiligten persönlichen Dateien.
  • ForgePlay hostet keine Installer für Apple GPTK, Steam, Microsoft Runtime, DirectX, NVIDIA PhysX, OpenAL oder XNA auf einem App-Server. Nutze offizielle Anbieter-Seiten oder bereits vorhandene Installer.
  • Fehlerberichte und Support-Anfragen können über GitHub Issues erstellt werden.

Guide d'assistance Français

Pour demander de l'aide, indiquez le nom du jeu, le Steam App ID si disponible, la version de macOS, le type de runner GPTK/Wine, le Prefix sélectionné, le dernier résultat de lancement et tout message d'erreur visible.

  • Si ForgePlay propose un dossier de support dans Diagnostics, créez-le d'abord et vérifiez les fichiers avant de les partager.
  • Un dossier de support peut contenir des journaux récents, diagnostics sélectionnés, métadonnées de Prefix, informations de compatibilité, listes de fichiers ignorés et résumés d'état de l'app.
  • N'envoyez pas de mots de passe Steam, codes Steam Guard, données de paiement, identifiants Apple ID, clés privées ni fichiers personnels sans rapport.
  • ForgePlay n'héberge pas les installateurs Apple GPTK, Steam, Microsoft Runtime, DirectX, NVIDIA PhysX, OpenAL ou XNA sur un serveur d'app. Utilisez les pages officielles ou les installateurs que vous possédez déjà.
  • Les rapports de bug et demandes d'assistance peuvent être ouverts via GitHub Issues.
