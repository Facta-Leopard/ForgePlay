ForgePlay Support

ForgePlay helps users set up local Windows game environments on Mac, review launch diagnostics, and create redacted support bundles when troubleshooting is needed.

Privacy Notice

ForgePlay does not include advertising tracking, third-party analytics, crash reporting SDKs, or telemetry SDKs. The app keeps setup, diagnostics, and support data local unless the user chooses to share files.

ForgePlay tracks Developer ID direct distribution and Mac App Store review as separate release paths. Direct distribution requires notarized Developer ID releases; App Store review requires App Store Connect submission, distribution signing, sandbox runtime QA, and Apple review. ForgePlay does not bundle Apple GPTK, Steam, Microsoft runtime installers, Wine distributions, or Windows game files.
