ForgePlay는 사용자의 Mac에서 Windows 게임 실행 환경을 설정하고, 실행 문제를 진단하고, 사용자가 요청할 때 지원 번들을 만들기 위해 설계되었습니다. ForgePlay 앱에는 광고 추적, 제3자 분석, 크래시 리포팅 SDK, 원격 텔레메트리 SDK가 포함되어 있지 않습니다.

  • 설정 상태, 선택한 파일 경로, 보안 범위 북마크, 게임 기록, Prefix 메타데이터, Runtime 설치 기록, 실행 기록, 진단 기록, 호환성 기록은 사용자의 Application Support 폴더에 저장됩니다.
  • 사용자가 선택한 게임 저장소, GPTK/Wine 실행 파일, Steam 설치 파일, Runtime 설치 파일, 로그, 가져온 게임 폴더에 접근할 수 있습니다. 접근 권한이 만료되면 다시 선택을 요청합니다.

  • ForgePlay는 Steam 비밀번호나 Steam Guard 코드를 요청, 저장, 전송하지 않습니다. Steam 로그인은 선택한 Wine/GPTK 환경 안의 Steam UI에서 진행됩니다.

Privacy Notice English

ForgePlay is built for local Windows game setup, launch diagnostics, and user-requested support bundle creation on the user's Mac. The app does not include advertising tracking, third-party analytics, crash reporting SDKs, or remote telemetry SDKs.

  • Setup state, selected paths, security-scoped bookmarks, game records, prefix metadata, runtime installation records, launch records, diagnostics, and compatibility records are stored in the user's Application Support folder.
  • ForgePlay can access game storage, GPTK/Wine runners, Steam installers, runtime installers, logs, and imported game folders selected by the user. If access expires, ForgePlay asks the user to select the item again.
  • AI diagnostics are off by default. When enabled, ForgePlay shows a redacted preview first and analyzes only with Apple Foundation Models on this Mac, without an external AI server or API key.
  • Support bundles are created only when the user requests them. ForgePlay redacts common secrets, Steam identifiers, home/user paths, and selected local storage or game paths where possible.
  • ForgePlay does not ask for, store, or transmit Steam passwords or Steam Guard codes. Steam login happens inside Steam's own UI in the selected Wine/GPTK environment.

Aviso de privacidad Español

ForgePlay está diseñado para configurar entornos locales de juegos de Windows en Mac, diagnosticar fallos de inicio y crear paquetes de soporte cuando el usuario lo solicita. La app no incluye seguimiento publicitario, analítica de terceros, SDK de informes de fallos ni telemetría remota.

  • El estado de configuración, rutas seleccionadas, marcadores con ámbito de seguridad, registros de juegos, metadatos de Prefix, instalaciones de Runtime, ejecuciones, diagnósticos y compatibilidad se guardan en Application Support del usuario.
  • ForgePlay accede solo a ubicaciones o archivos seleccionados por el usuario, como almacenamiento de juegos, ejecutores GPTK/Wine, instaladores de Steam o Runtime, registros y carpetas de juegos importadas.
  • El diagnóstico con IA está desactivado de forma predeterminada. Si se activa, ForgePlay muestra primero una vista previa redactada y usa solo Apple Foundation Models en este Mac, sin servidor externo ni clave API.
  • Los paquetes de soporte se crean solo a petición del usuario y redactan secretos comunes, identificadores de Steam, rutas de usuario y rutas locales seleccionadas cuando sea posible.
  • ForgePlay no solicita, almacena ni transmite contraseñas de Steam o códigos de Steam Guard. El inicio de sesión ocurre en la interfaz de Steam dentro del entorno Wine/GPTK seleccionado.

Datenschutzhinweis Deutsch

ForgePlay dient zur lokalen Einrichtung von Windows-Spielumgebungen auf dem Mac, zur Startdiagnose und zur Erstellung von Support-Paketen auf Wunsch des Nutzers. Die App enthält kein Werbetracking, keine Drittanbieter-Analyse, kein Crash-Reporting-SDK und keine Remote-Telemetrie.

  • Einrichtungsstatus, ausgewählte Pfade, Security-Scoped Bookmarks, Spieleinträge, Prefix-Metadaten, Runtime-Installationen, Startprotokolle, Diagnosen und Kompatibilitätsdaten werden im Application-Support-Ordner des Nutzers gespeichert.
  • ForgePlay greift auf vom Nutzer ausgewählte Spielordner, GPTK/Wine-Runner, Steam-Installer, Runtime-Installer, Logs und importierte Spielordner zu. Bei abgelaufenem Zugriff wird eine erneute Auswahl verlangt.
  • KI-Diagnosen sind standardmäßig deaktiviert. Nach Aktivierung zeigt ForgePlay zuerst eine geschwärzte Vorschau und nutzt nur Apple Foundation Models auf diesem Mac, ohne externen KI-Server oder API-Schlüssel.
  • Support-Pakete werden nur auf Anforderung erstellt und schwärzen nach Möglichkeit häufige Geheimnisse, Steam-Kennungen, Benutzerpfade und ausgewählte lokale Speicher- oder Spielpfade.
  • ForgePlay fragt keine Steam-Passwörter oder Steam-Guard-Codes ab, speichert sie nicht und überträgt sie nicht. Die Anmeldung erfolgt in Steams eigener UI im ausgewählten Wine/GPTK-Umfeld.

Avis de confidentialité Français
