Privacy Policy

ForgePlay Privacy Policy

One page, multiple languages. No language subpaths are required for GitHub Pages setup.

Effective date: 2026-06-22

Short Version

ForgePlay is designed as a local-first Mac app. It does not require an account, does not use an external AI endpoint for diagnostics, and does not include or host third-party game runtimes or installers. Optional support bundles and diagnostics are created only when the user chooses to use them.

English

Privacy Policy

ForgePlay helps users prepare local Windows game execution environments on macOS. The app stores configuration and diagnostic information locally on the user’s Mac so the user can manage game setup, runtime status, compatibility notes, and troubleshooting history.

Information stored locally

  • Selected ForgePlay storage location, selected GPTK/Wine runner path, Steam installer path, game library paths, prefix records, runtime installation records, compatibility recipes, launch records, and diagnostic records.
  • Security-scoped bookmarks may be stored so the app can reopen user-selected files or folders after permission is granted by the user.
  • Launch logs and diagnostic records may be stored locally for troubleshooting and support bundle creation.

What ForgePlay does not collect

  • ForgePlay does not require a ForgePlay account.
  • ForgePlay does not collect Steam credentials, Steam Guard codes, payment information, or gameplay activity for a ForgePlay server.
  • ForgePlay does not send AI diagnostic logs to an external AI provider.

AI diagnostics

AI diagnostics are off by default. When enabled by the user, ForgePlay first shows a redacted preview, then uses Apple Foundation Models on this Mac when available. The app is designed to redact common secrets, API keys, bearer tokens, passwords, Steam identifiers, home paths, and selected local paths before the optional local AI diagnostic prompt is generated.

Support bundles

Support bundles are created only when requested by the user. ForgePlay attempts to redact sensitive values before saving a bundle. The user decides whether and where to share the generated bundle.

Network access

ForgePlay may open official third-party download pages at the user’s request and may check a signed compatibility database feed if the user configures that feature and the app includes a trusted public key. ForgePlay does not host Microsoft, NVIDIA, OpenAL, Steam, Wine, or Apple GPTK installers.

한국어

개인정보 처리방침

ForgePlay는 macOS에서 Windows 게임 실행 환경을 준비하도록 돕는 로컬 중심 앱입니다. 앱은 게임 설정, 실행 환경 상태, 호환성 정보, 문제 해결 기록을 사용자가 관리할 수 있도록 필요한 정보를 사용자의 Mac에 저장합니다.

로컬에 저장되는 정보

  • 사용자가 선택한 ForgePlay 저장 위치, GPTK/Wine 실행 파일 경로, Steam 설치 파일 경로, 게임 라이브러리 경로, Prefix 기록, Runtime 설치 기록, 호환성 레시피, 실행 기록, 진단 기록.
  • 사용자가 권한을 부여한 파일 또는 폴더를 다시 열기 위해 security-scoped bookmark가 저장될 수 있습니다.
  • 문제 해결과 지원 번들 생성을 위해 실행 로그와 진단 기록이 로컬에 저장될 수 있습니다.

ForgePlay가 수집하지 않는 정보

  • ForgePlay 계정은 필요하지 않습니다.
  • Steam 자격 증명, Steam Guard 코드, 결제 정보, 게임 플레이 활동을 ForgePlay 서버로 수집하지 않습니다.
  • AI 진단 로그를 외부 AI 제공자에게 보내지 않습니다.

AI 진단

AI 진단은 기본값이 꺼짐입니다. 사용자가 켠 경우 ForgePlay는 먼저 가려진 미리보기를 보여준 뒤, 사용 가능한 경우 이 Mac의 Apple Foundation Models를 사용합니다. 선택적 로컬 AI 진단 프롬프트를 만들기 전에 일반적인 secret, API key, bearer token, password, Steam 식별자, 홈 경로, 사용자가 선택한 로컬 경로를 가리도록 설계되어 있습니다.

지원 번들

지원 번들은 사용자가 요청한 경우에만 생성됩니다. ForgePlay는 번들을 저장하기 전에 민감한 값을 가리려고 시도합니다. 생성된 번들을 공유할지와 어디에 공유할지는 사용자가 결정합니다.

네트워크 접근

ForgePlay는 사용자가 요청할 때 공식 외부 다운로드 페이지를 열 수 있고, 사용자가 기능을 설정했으며 앱에 신뢰할 수 있는 공개키가 포함된 경우 서명된 호환성 DB feed를 확인할 수 있습니다. ForgePlay는 Microsoft, NVIDIA, OpenAL, Steam, Wine, Apple GPTK 설치 파일을 호스팅하지 않습니다.

Política de privacidad

ForgePlay es una app local-first para ayudar a preparar entornos de ejecución de juegos de Windows en macOS. La app guarda la configuración y la información de diagnóstico localmente en el Mac del usuario.

Información almacenada localmente

  • Ubicación de almacenamiento de ForgePlay, ruta del runner GPTK/Wine, ruta del instalador de Steam, rutas de bibliotecas de juegos, registros de prefix, runtimes, recetas de compatibilidad, lanzamientos y diagnósticos.
  • Puede guardar marcadores con alcance de seguridad para volver a abrir archivos o carpetas seleccionados por el usuario.
  • Los registros de ejecución y diagnósticos pueden guardarse localmente para solucionar problemas y crear paquetes de soporte.

Lo que ForgePlay no recopila

  • No se requiere una cuenta de ForgePlay.
  • No recopila credenciales de Steam, códigos de Steam Guard, datos de pago ni actividad de juego para un servidor de ForgePlay.
  • No envía registros de diagnóstico de IA a un proveedor externo de IA.

Diagnóstico con IA

El diagnóstico con IA está desactivado de forma predeterminada. Si el usuario lo activa, ForgePlay muestra primero una vista previa redactada y luego usa Apple Foundation Models en este Mac cuando esté disponible.

Paquetes de soporte

Los paquetes de soporte se crean solo cuando el usuario los solicita. ForgePlay intenta redactar valores sensibles antes de guardar el paquete, y el usuario decide si lo comparte.

Acceso de red

ForgePlay puede abrir páginas oficiales de descarga cuando el usuario lo solicita y puede comprobar una base de datos de compatibilidad firmada si la función está configurada. ForgePlay no aloja instaladores de Microsoft, NVIDIA, OpenAL, Steam, Wine ni Apple GPTK.
